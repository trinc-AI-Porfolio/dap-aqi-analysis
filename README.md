# 🌍 Phân Tích Chất Lượng Không Khí (AQI) — DAP Final Project

Dự án cuối kỳ môn **Data Analytics & Programming** — Phân tích và dự báo chỉ số AQI (Air Quality Index) sử dụng Python, Machine Learning và Streamlit Dashboard tích hợp Gemini AI chatbot.

## 📊 Tính Năng
- 📈 Phân tích dữ liệu AQI — xu hướng, phân phối, tương quan theo thời gian
- 🤖 Mô hình ML — train và đánh giá mô hình dự báo AQI
- 🖥️ Dashboard Streamlit — trực quan hóa tương tác
- 💬 Chatbot Gemini AI — hỏi đáp dữ liệu bằng ngôn ngữ tự nhiên

## 🛠️ Tech Stack
Python · Streamlit · Pandas · Scikit-learn · Matplotlib · Google Gemini API · SQLite

## 🚀 Cách Chạy
```bash
pip install -r requirements.txt
streamlit run streamlit.py
```

> Cần tạo file `.env` với `GEMINI_API_KEY=your_key`
